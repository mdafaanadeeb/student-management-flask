
# Student Management System (Flask + SQLAlchemy)
## 🚀 Live Demo
https://student-management-flask-2.onrender.com/login?next=%2F

## Overview
A full-stack web-based Student Management System built using Flask, SQLAlchemy, and Flask-Login.

## Features
- User Authentication (Register/Login)
- Add/Delete Students
- Search & Sort
- GPA Analytics
- AI Performance Classification
- Role-based Ready Architecture
- Production Deployment with Gunicorn

## Tech Stack
- Python
- Flask
- SQLAlchemy
- Flask-Login
- Bootstrap
- SQLite
- Gunicorn

## Architecture
Application Factory Pattern
Blueprint Routing
ORM-based Database Modeling

## How to Run Locally
1. Create virtual environment
2. pip install -r requirements.txt
3. python run.py
